Description

This is a Python console-based calculator that allows users to perform basic arithmetic operations. Users can:

Add, subtract, multiply, and divide numbers

Continue calculations using the previous result

Perform the first operation with two numbers

The program validates user input to prevent errors such as invalid numbers or division by zero.
